# VRL for Zed

[Vector Remap Language (VRL)](https://vrl.dev) extension for Zed.

## Features

- Syntax highlighting
- Auto indentation
- Brackets matching
- Comment and RegExp injection in VRL files
- Redactions

## Tree-Sitter Grammar

See [tree-sitter-vrl](https://github.com/belltoy/tree-sitter-vrl).
